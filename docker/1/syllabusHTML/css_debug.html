Introduction

Nous avons vu des outils permettant de valider aussi le code HTML que CSS. Cependant il également utile de disposer d'un outil de debugging afin par exemple de comprendre pourquoi un style ne s'applique pas sur une balise.

Les navigateurs Firefox et Chrome disposent tous les 2 d'un module destiné au développement. Activez-le et vous pourrez facilement voir quel style s'applique sur telle balise ou encore voir le résultat de votre site sur un écran de la taille d'un SmartPhone.
